# Quick Start

## `Basic configuration`

1. Install [Node.js](https://nodejs.org/en/)
2. Install server and client packages:
   ```
   npm install
   ```
3. Follow .env.example for client and server side and create your own

4. Type the following in the command prompt:

   ```
   npm run dev
   ```
